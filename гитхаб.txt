Для PyCharm порядок такой:
1. Клонировать проект

В терминале PyCharm или обычном PowerShell:

git clone https://github.com/kup1tman/Demo.git
cd Demo

Можно и без .git, обычно оба варианта работают:

git clone https://github.com/kup1tman/Demo
2. Создать виртуальное окружение
python -m venv .venv
3. Выбрать .venv в PyCharm

В PyCharm:

File → Settings → Project: Demo → Python Interpreter

Дальше:

Add Interpreter → Add Local Interpreter → Existing

Выбери файл:

Demo\.venv\Scripts\python.exe

То есть не просто папку .venv, а именно python.exe внутри неё.

4. Открыть новый терминал в PyCharm

После выбора интерпретатора закрой старый терминал и открой новый.

В терминале должно быть примерно так:

(.venv) PS C:\...\Demo>

Если видишь (.venv) — всё правильно.

5. Установить зависимости
pip install pyside6 pandas openpyxl psycopg2

Но есть важный момент: psycopg2 на Windows иногда не ставится, потому что требует сборку. Если будет ошибка, используй:

pip install psycopg2-binary

То есть полный вариант:

pip install pyside6 pandas openpyxl psycopg2-binary
Для VS Code отличие только в выборе интерпретатора

Там да:

Ctrl + Shift + P → Python: Select Interpreter → .venv

А в PyCharm это делается через настройки проекта.

Итого: команды git clone, python -m venv .venv, pip install ... работают и в PyCharm, но выбор интерпретатора делается через Settings → Python Interpreter.

PyCharm:
1. Настроить Git (один раз)
bashgit config --global user.name "kup1tman"
git config --global user.email "kup111pitman@gmail.com"
2. Создать репозиторий на GitHub
Зайти на github.com → кнопка «New» → задать имя → Create repository. Не добавлять README и .gitignore при создании, если проект уже существует.
3. В терминале PyCharm (View → Tool Windows → Terminal) — инициализировать локальный репозиторий
bashgit init
4. Добавить все файлы
bashgit add .
5. Сделать первый коммит
bashgit commit -m "first commit"
6. Переименовать ветку в main
bashgit branch -M main
7. Привязать удалённый репозиторий
bashgit remote add origin https://github.com/ВашЛогин/ИмяРепозитория.git
Эту ссылку GitHub показывает сразу после создания репозитория.
8. Отправить проект
bashgit push -u origin main
GitHub попросит войти — нужен Personal Access Token вместо пароля (Settings → Developer settings → Tokens → Generate new token, отметить галочку repo).
